This page explains how my Flowise AI project operates, how it is hosted, and why it may go offline when my system shuts down.

1. Project Overview

Flowise RAG Local AI Cloudflare Tunnel

This Flowise project is built using a Retrieval-Augmented Generation (RAG) pipeline. It uses DOCX/PDF files uploaded to Flowise, which are transformed into embedding vectors and stored in a local vector store.

The agent is powered by Groq / local LLMs and connected with tool-calling and retrievers for accurate answers on my trading or knowledge datasets.

2. Hosting (Cloudflare Tunnel)

This chatbot is NOT hosted on a cloud server. Instead, it runs **locally on my PC**, and I expose it to the internet using:

Cloudflare Tunnel
Cloudflare Tunnel creates a secure public URL that connects to my local Flowise server. It allows anyone to access the chatbot online without paying for hosting.

Since the tunnel links directly to my computer, the entire system depends on my local machine being ON.
